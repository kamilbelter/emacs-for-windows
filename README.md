# emacs-for-windows
Emacs configuration optimized for windows

# Fonts
Default font is "Source Code Pro".

# Comments
Configuration are organized in init files located in ~/.emacs.d/elisp as in M-Emacs. I'm using newest Emacs 27.1 from msys2 project.
